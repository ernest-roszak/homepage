Znaczniki HTML

Ta sekcja poświęcona jest znacznikom html

  • <br/> --> enter - nowa linia
  • <b> --> bold - pogróbienie (nie ma zanczenia semnatyznego - tylko wizualnie)
  • <i> --> italic - kursywa
  • <strong> --> tez pogróbienie, ale stosujemy w celu pokreślenia jakiegoś waznego, wysoka istotność powazność tekstu
  • <em> --> podkresla ze tekst moze miec inne znaczenie np. irona (alez ja kocham koty hahahah
  • <a --> linki na zewnątrz i wewnatrz dokumentu
  • href --> adres domentu zewnetrznego np. https://
  • id="nazwa" --> przypisanie id do sekcji ( stawianie kotwicy mozna potem to odszukać a hrefem )
  • href="#nazwa" --> odnosnik wewnątrz dokumetu ( odszukanie kotwicy ) znak # jest SUPER istotny
  • pisać dokładnie do czego prowadzi link we wszystkich odnosnikach a (np. wróć na górę strony <header id= "top"> )
  • <nav> --> navigacja strony
  • <img src="ściezka_do_zdjecia" alt="opis"/> --> umieszczanie grafik

Specyficzność w css

Kolejność stylowania według wyzszości

  • !important --> zabija (nadpisuje) wszystkie inne reguły
  • (!important, style ) , #name , .name , elemnt
  • Przykład 1 body h1 {
    color: yellow;
    } dwa elementy body + h1 - wazniejsze od np. samego h1
  • Przykład 2 #main-header {
    color: red;
    } ID wyzsza specyficzność od html body h1 {color: yellow} --> ID większe od 3 elementów
  • Kiedy specyficzność jest równa wygrywa style blizszy (nizej) do elementu stylowanego

Rekomenduje dodawać wszystkich elemnetów i unikać zagniezdzeń (nie pisać header nav a tylko użyć klasy np header-link) - Tworzymy klasy nawet jeśli mamy tylko jedne elemment
